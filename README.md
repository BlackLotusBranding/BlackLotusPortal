# Black Lotus Employee Portal v1.0

Official mobile-first Employee Portal for Black Lotus Branding.

## Upload to GitHub Pages
Upload the contents of this folder to the `BlackLotusEmployeePortal` repository, replacing the existing files. Enable GitHub Pages from Settings → Pages → Deploy from branch → main → root.

## Included
- Mobile and desktop homepage
- Scout Academy training materials
- Downloads Library
- Google Workspace Hub
- Forms Center
- Black Lotus graphics and branding
